3)What are the basic building blocks of an HTML document?
Name Tag Description
Doctype declaration: <!DOCTYPE html> Defines the document type and HTML version we're using.Here is HTML version 5
HTML Element: <html> The root element that wraps all the content on the page
Head: <head> Contains meta-information about the document, such as the title and links to stylesheets.
Title: <title> Sets the title of the document, which appears in the browser’s title bar.
Body: <body> Contains the content of the HTML document, such as text, images, and links.
